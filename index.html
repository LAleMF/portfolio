PayFlow

Divide gastos, simplifica cuentas.

Aplicación web full-stack para gestionar y repartir gastos en grupo. Incluye registro/login, grupos, gastos compartidos, cálculo automático de balances, reembolsos y pagos online con Stripe. Desplegada en producción con arquitectura MVC en PHP.

  • Demo en vivo en hosting real
  • Integración de pagos con Stripe (test mode)

Proyecto DAM · Android + Servidor Java

Billsapp

Gestión de gastos en tu bolsillo.

App Android nativa para gestionar gastos compartidos en grupo. Arquitectura cliente-servidor por sockets TCP: cliente Kotlin en Android y servidor Java con Oracle. Funcionalidades completas: grupos, gastos, balances, reembolsos y autenticación segura con BCrypt.

  • App Android nativa en Kotlin con Material Design
  • Servidor Java multihilo con sockets TCP
  • Persistencia en Oracle con PL/SQL y triggers
Kotlin Java Android Oracle Sockets TCP Maven
